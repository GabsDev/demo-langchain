# Demo de Sistemas de Inteligencia Artificial con LangChain y LangGraph 🚀

Este repositorio presenta un **demo profesional** de cómo construir sistemas avanzados de Inteligencia Artificial aplicados a entornos empresariales. El proyecto integra **Python, LangChain y LangGraph** para mostrar cómo crear agentes inteligentes capaces de procesar información, tomar decisiones autónomas y resolver problemas complejos del mundo real.

---

## 🎯 Objetivo del Demo

Demostrar la capacidad de diseñar y desplegar arquitecturas de IA modernas que:
- Integran múltiples fuentes de datos.
- Mantienen memoria persistente y contexto a largo plazo.
- Ejecutan tareas complejas de forma autónoma.
- Escalan hacia sistemas multi-agente especializados.

Ejemplos incluidos:
- **Asistente legal** que analiza contratos automáticamente.
- **Helpdesk inteligente** que escala a humanos cuando es necesario.
- **Centro de operaciones de seguridad** con agentes especializados.

---

## 📚 Componentes principales

- **[Fundamentos de LangChain](ca://s?q=Fundamentos_de_LangChain)**: Runnables, LCEL, Prompt Templates, Output Parsers, Pydantic.
- **[Sistemas RAG](ca://s?q=Sistemas_RAG)**: Document Loaders, Text Splitters, Embeddings, bases vectoriales (ChromaDB, FAISS).
- **[LangGraph avanzado](ca://s?q=Dominar_LangGraph)**: control de flujo, estados, checkpoints, persistencia.
- **[Memoria inteligente](ca://s?q=Memoria_en_agentes_de_IA)**: conversación, vectorial, ventanas deslizantes, memoria persistente.
- **[Agentes especializados](ca://s?q=Agentes_de_IA_especializados)**: integración de APIs externas, herramientas personalizadas, multi-agente.
- **[Aplicaciones web](ca://s?q=Aplicaciones_web_con_Streamlit_y_FastAPI)**: dashboards interactivos con Streamlit y APIs con FastAPI.
- **[Casos empresariales](ca://s?q=Casos_de_uso_empresariales_en_IA)**: evaluación de candidatos, asistentes legales, helpdesks, centros de operaciones.
